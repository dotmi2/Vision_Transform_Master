# 📷 Vision Transform Master

一款专业、高颜值的工业相机标定与逆透视变换桌面端软件。

## ✨ 核心特性

* **🪄 现代化 UI**：采用 Fluent Design 风格的深色主题，告别传统的简陋界面。
* **🎯 精准相机标定**：支持导入 ChArUco 标定板图片，全自动提取角点，计算内参矩阵 ($K$) 与畸变系数 ($D$)。
* **📐 逆透视变换**：可视化交互选点，支持极速多线程打表（生成去畸变+逆透视的级联映射矩阵）。
* **💾 资产管理系统**：内置本地轻量级数据库，标定参数与逆透视配置一键保存、一键恢复，支持导出 YAML 供 Python或C/C++ 端直接调用。

## 🚀 快速体验

如果你只是想使用本软件，无需配置 Python 环境！
1. 前往右侧的 **[发布]** 页面。
2. 下载最新版本的 `Vision_Transform_Master_v1.0.zip`。
3. 解压后，双击运行 `image_process_gui.exe` 即可。
